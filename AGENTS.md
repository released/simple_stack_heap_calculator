# AGENTS.md

## Purpose

This file is the project-specific implementation baseline for `simple_stack_heap_calculator`.
If any old notes conflict with this file, this file wins for this project.

## Platform / Build Baseline

- OS: Windows 10/11 x64
- Language: C++17
- UI: MFC native EXE
- Runtime: static CRT / static MFC
- Delivery goal: zero-install EXE on target Windows PCs
- No compiler-specific external driver dependency is required to open the tool itself

## Product Goal

- Parse embedded linker map files
- Detect map format from content instead of filename extension
- Estimate:
  - `Code`
  - `RO-data`
  - `RW-data`
  - `ZI-data`
  - total flash usage
  - static RAM usage
  - stack / heap reserved or estimated size
  - estimated RAM required
  - RAM remaining and risk level

## Supported Formats

Detected formats:

- `KEIL_ARMCC_ARMCLANG`
- `GCC_GNU_LD`
- `IAR_ILINK`
- `RENESAS_CC_RL`
- `RENESAS_CC_RH`
- `GHS`
- `UNKNOWN`

Current parser implementation status:

- `KEIL`: implemented
- `GCC`: implemented
- `IAR`: implemented
- `RL78 (CC-RL)`: partial parsing
- `RH850 (CC-RH)`: partial parsing
- `GHS`: detection only

## Project Layout

- `src\ui\`
  - main view layout, control state, result presentation
- `src\core\`
  - analysis flow, report calculation, parser selection policy
- `src\detector\`
  - weighted map-format detection
- `src\parser\`
  - compiler-specific parsing modules
- `src\model\`
  - shared data structures and enums
- `src\util\`
  - string, file, and formatting helpers
- `src\generated\`
  - build version header generated by script
- `scripts\`
  - build / clean / version helper scripts
- `build\`
  - output EXE and intermediate artifacts
  - portable user config INI

## UI Baseline

- Map path edit + browse button
- Build version and build date display
- Automatic detected-format display
- Manual parser selection by checkbox:
  - `KEIL`
  - `IAR`
  - `GCC`
  - `RL78`
  - `RH850`
- RAM limit input
- Parse button
- Analysis log area
- Analysis summary area
- Metrics table
- Bottom stack / heap recommendation table labeled `reserved / estimated`

## Script Baseline

- `scripts\build.ps1`
  - defaults to `Release|x64`
  - closes any running tool instance before build
  - launches the built EXE unless `-NoRun` is used
- `scripts\build_release.bat`
  - builds `Release|x64`
  - closes any running tool instance before build
  - relaunches the EXE after success
- `scripts\clean_build.bat`
  - removes build artifacts
  - keeps only `build\SimpleStackHeapCalculator.exe` when present
- `scripts\update_build_version.ps1`
  - increments `version.counter`
  - regenerates `src\generated\build_version.h`

## Portable Config

- Config file: `build\SimpleStackHeapCalculator.ini`
- Config is created automatically when missing
- Current persisted fields:
  - last map path
  - RAM limit text
- Manual parser always defaults to `KEIL` on startup
- If auto-detection disagrees with manual selection, the detected checkbox is highlighted with `[Auto]`

## Current Constraints

- `RL78` and `RH850` parsing are not yet verified against a broad sample set
- `GHS` currently supports detection only, not full parsing
